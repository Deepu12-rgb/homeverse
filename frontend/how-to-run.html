How to Run Homeverse

Note: Some features of Homeverse require a running server to work properly. Opening the HTML files directly in your browser will result in limited functionality.

Option 1: Running with Node.js (Recommended)

1 Install Node.js

If you don't have Node.js installed, download and install it from nodejs.org.

2 Install MongoDB

Install MongoDB from mongodb.com or use MongoDB Atlas.

3 Install Dependencies

Open a terminal/command prompt in the project root directory and run:

npm run install-all
4 Start the Application

In the same terminal, run:

npm start

This will start both the backend and frontend servers.

5 Access the Application

Open your browser and navigate to:

http://localhost:3000

Option 2: Running Frontend Only (Limited Functionality)

1 Open index.html

Simply open the index.html file in your browser.

Note that features requiring the backend (login, signup, profile management) will not work.

Troubleshooting

Common Issues:

  • API Connection Errors: Make sure the backend server is running on port 5000.
  • MongoDB Connection Errors: Ensure MongoDB is running and accessible.
  • Port Already in Use: If port 3000 or 5000 is already in use, modify the port in the respective server files.
Need Help? If you're still having issues, please check the README.md file or contact support.
